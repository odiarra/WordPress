##**English**
-----------

>**I dedicate this plugin to my beloved one, the unique that my heart beats for her: Asmae. I LOVE YOU!**

**Lys Creation Multi-Video Thumbnails Widgets** is a free plugin and open source software that automatically insert Video thmbnails from different sources (Google, Youtube, Yahoo, Vimeo, mySapce, DailyMotion). This plugin is compatible with the widget system.

Adds a sidebar widget to display videos thumbnails of your own choice from DailyMotion, Vimeo, Google, Myspace and YouTube.

You have full control over the Videos. You determine how many Widgets you want to show.

Syntax: [myspace, google, youtube, vimeo, dailymotion]@[ID]@[titre]@[categorie (dailymotion)*](Line Brake) whithout the brackets. * Is optional. 
Do not add a (Line Brake) after the last video in the list.

The SEPERATOR "@" must be in to devide the chain into:

	- Source (put it in lower)
	- ID
	- Title
	- Category "used for DailyMotion ONLY"

**[Download](http://downloads.wordpress.org/plugin/multi-video-thumbnail-sources-widget.zip)**

*Features:*
-----------

* Automatically insert Video thmbnails from different sources (Google, Youtube, Yahoo, Vimeo, mySapce, DailyMotion) in your Website/blog.
* **Get easily Videos** with this free plugin.
* The plugin is compatible with the widget system.
* Friendly and intuitive interface.
* You're always on the control.
* Choose if you want to display per page, and per post.

== Frequently Asked Questions ==

**English**
-----------

= Can I use this plugin to only display Lys Creation Multi Video Thumbnails? =

Sure, you can use this FREE plugin to display Videos from different sources. Remember with **Lys Creation Multi Video Thumbnails (Plugin) widget** you always keep control.

= What is Lys Creation Multi Video Thumbnails? =

Lys Creation Multi Video Thumbnails is a free program that allows online publishers to get videos thumbnails and display Videos with ShadowBox (ShadowBox is included in this plugin).

= How do I display video thumnails with Lys Creation Multi Video Thumbnails? =

You just need to know and have: 

* *The Source*: youtube for example
* *The Video ID*: http://www.youtube.com/watch?v=**lT-MeHJhtEc** (take here what follows `*?v=*` so it's **lT-MeHJhtEc**
* *The Title*: We have *Joe Ashkar - Habibet Alby*
* *The Category*: This is used only with *Dailymotion*

Now: We past all together by using the SEPERATOR **@** so we'll get youtube@lT-MeHJhtEc@Joe Ashkar - Habibet Alby@No comment

= The plugin does not work! =

Did you follow the step above? If so, it should be ok, try just one video to know if it's correctly gotten.
Some videos can NOT be displayed because of the source: NOT allowed to be shown only from it's site.

= I got this error: Warning: array_merge() [function.array-merge]: Argument #2 is not an array in ...../wp-includes/widgets.php on line 777 =

Unfortunately your Theme got an error/issue, try to contact the developper to resolve it. Now you have to desactivate this plugin, you can NOT use it on this theme.



**Français**
------------

= Puis-je utiliser ce plugin pour afficher des miniatures avec Lys Creation Multi Video Thumbnails? =

Sûrement, vous pouvez utiliser ce plugin GRATUIT pour afficher des Vidéos depuis différentes sources. Rappelez-vous qu'avec **Lys Création Multi Video Thumbnails (Plugin) widget** vous maintenez le contrôle des Vidéos.

= Qu'est-ce que Lys Creation Multi Video Thumbnails? =

Lys Creation Multi Video Thumbnails est un programme gratuit qui permet aux éditeurs en ligne d'obtenir des miniatures de vidéos et les afficher avec ShadowBox (Shadowbox est inclu dans le plugin).

= Comment puis-je afficher des miniatures avec Lys Creation Multi Video Thumbnails? =

Vous aurez besoin seulement de connaître et avoir: 

* *La Source*: youtube par exemple
* *L'ID de la Vidéo*: http://www.youtube.com/watch?v=**lT-MeHJhtEc** (Notez ici ce qui suit `*?v=*` donc on a **lT-MeHJhtEc**
* *Le Titre*: Nous avons *Joe Ashkar - Habibet Alby*
* *La Catégorie*: Celle-là est utilisée selement avec *Dailymotion*

Maintenant: Nous allons coller tous ensemble en utilisant le SEPARATEUR **@** et nous obtenons youtube@lT-MeHJhtEc@Joe Ashkar - Habibet Alby@No comment

= Le plugin ne fonctionne pas! =

Avez-vous suivi les étapes ci-dessus? Si Oui, cela doit bien fonctionner. Essayez avec une seule vidéo pour savoir.
Certaines plate-formes (sites de média) intérdisent l'affichage externe de vidéos.

= J'ai obtenu cette erreur: Warning: array_merge() [function.array-merge]: Argument #2 is not an array in ...../wp-includes/widgets.php on line 777 =

Malheureusement le code de votre thème (template) a des erreurs, essayez de contacter le développeur de votre template pour le résoudre. Maintenant, vous devez désactiver cette extension, vous NE pouvez pas l'utiliser sur ce thème.
